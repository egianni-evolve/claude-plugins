Manage OKRs using the okr-manager skill.

$ARGUMENTS

If no arguments: Show current quarter's OKRs with key results, progress percentages, and status indicators.

If "create": Walk the user through creating a new objective with key results for the current quarter.

If "update [KR] to [value]": Update the specified key result's progress, recalculate percentage, and update status based on time elapsed in the quarter.

Keep responses concise with clear progress indicators.
