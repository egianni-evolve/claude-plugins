Create a task in the user's Notion Tasks database using the task-creator skill.

$ARGUMENTS

Set Status to "To Do". Set Created Date to today. Infer priority and tags from the content. If a due date is mentioned, set it. If an area or key result is mentioned, link it.

Confirm with a single short line. Do not explain what you did - just confirm the task was created.
