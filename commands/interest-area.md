Manage areas of interest using the area-tracker skill.

$ARGUMENTS

If an area name is provided: Create a new area with that name. Set Status to "Idea Stage". Infer categories from the name. Confirm briefly.

If "list": Show all areas grouped by status (Active Exploration, Learning, Watching, Idea Stage). Include count of linked brain dumps and tasks.

If "active": Show only areas in "Active Exploration" or "Learning" status with linked item counts.

Keep responses concise.
